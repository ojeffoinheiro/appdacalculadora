# How to use :fire:
## Precondition :white_flag:
You should have:

- NodeJS
- NPM

## Clone this repository

```bash
$ git clone https://github.com/ojeffoinheiro/tasks.git
```

## Install the dependencies

```bash
cd backend && npm i
cd mobile && npm i
```


## Running the Server üåê

Enter on folder

```bash
$ cd backend
```

Run the server

```bash
$ yarn start
```

## Run on mobile :iphone:

Entre na pasta

```bash
$ cd mobile
```

Run the application
```bash
$ react-native run-android
```
